# Exchange Rate App

Este repositorio contiene información funcional y guía de cómo instalar la aplicación en un ambiente local.

## Tecnologías

* Java 11.
* Spring Framework.
* Spring Boot.
* Jwt.
* Postgresql.
* Angular.
* Docker.

## Ubicación de información funcional

* Arquitectura de Software.
* Análisis de rendimiento por Jmeter.
* Colecciones Postman.
